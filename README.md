\# Expense Tracker



Aplicación web para registrar y visualizar gastos personales.



\## Funcionalidades



\- Agregar gastos con descripción y monto

\- Eliminar gastos

\- Visualizar total acumulado

\- Persistencia mediante localStorage



\## Tecnologías



\- HTML

\- CSS

\- JavaScript



\## Ejecución



Abrir index.html en el navegador.



\## Estructura



\- index.html: estructura de la interfaz

\- style.css: estilos visuales

\- app.js: lógica de la aplicación



\## Objetivo



Practicar manipulación del DOM, almacenamiento local y organización de código en frontend.
