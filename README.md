# Expense Tracker

Aplicación web para la gestión de gastos personales, desarrollada con HTML, CSS y JavaScript. Permite registrar, visualizar y eliminar gastos, manteniendo la información almacenada en el navegador mediante almacenamiento local.

## Descripción

Este proyecto consiste en una aplicación frontend simple orientada a la práctica de desarrollo web. Implementa funcionalidades básicas de un sistema de seguimiento de gastos, reforzando conceptos como manipulación del DOM, manejo de eventos y persistencia de datos en el cliente.

## Funcionalidades

- Registro de gastos con descripción y monto
- Visualización de lista de gastos
- Eliminación de registros individuales
- Cálculo automático del total acumulado
- Persistencia de datos mediante localStorage

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

## Ejecución

1. Clonar el repositorio:

git clone https://github.com/Ramiro-Silva/expense-tracker.git

2. Acceder a la carpeta del proyecto:

cd expense-tracker

3. Abrir el archivo `index.html` en el navegador.

No se requieren dependencias ni instalación adicional.

## Uso

1. Ingresar una descripción del gasto  
2. Ingresar el monto correspondiente  
3. Presionar el botón "Agregar"  
4. Visualizar el listado actualizado y el total acumulado  
5. Eliminar gastos según sea necesario  

## Consideraciones técnicas

- Los datos se almacenan en el navegador mediante localStorage  
- No requiere conexión a internet  
- No utiliza frameworks ni librerías externas
